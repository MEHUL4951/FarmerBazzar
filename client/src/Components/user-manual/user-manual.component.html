6. AI Assistant

The AI Assistant helps users by answering queries and providing intelligent responses. It also saves user history for future reference.

1. Using the AI Assistant

  • Navigate to the "AI Assistant" page from the navigation bar.
  • Enter your query in the input box.
  • Click on the Submit button to get an instant response.

2. Viewing and Managing Query History

All queries and responses are saved as history, allowing users to revisit past interactions.

  • The history appears below the query input box.
  • Users can scroll through previous conversations.

3. Clearing Query History

  • Click the Clear History button to remove all past queries.
  • This action is irreversible and permanently deletes saved queries.

4. Edge Cases & Troubleshooting

No Response from AI?

  • Check your internet connection.
  • Try refreshing the page and asking again.

Query Not Saved in History?

  • Ensure that history saving is enabled.
  • Wait a few seconds and check again.

Unable to Clear History?

  • Ensure you have query history available.
  • Try clicking the Clear History button again.

8. Fertilizer Recommendation

The Fertilizer Recommendation feature helps farmers determine the best fertilizer based on crop type, soil, and climate conditions.

1. Accessing the Fertilizer Recommendation

  • Navigate to the Dashboard > Fertilizer Recommendation section.
  • The form for fertilizer recommendations will be displayed.

2. Entering Required Details

  • Crop Name: Enter the name of the crop (e.g., Wheat, Rice, Corn).
  • Soil Type: Select from the dropdown (e.g., Sandy, Clay, Loam).
  • Soil pH: Enter the pH level of the soil.
  • Climate Condition: Select the current climate condition (e.g., Dry, Humid, Wet).

3. Getting the Fertilizer Recommendation

  • Click the Get Recommendation button.
  • The system processes the data and suggests the best fertilizer for the selected conditions.

4. Edge Cases & Troubleshooting

No Recommendation Provided?

  • Ensure all fields are correctly filled.
  • Check if the soil pH value is within a valid range (3.5 - 9.0).

Incorrect or Unexpected Recommendation?

  • Try re-entering the details and submitting again.
  • Ensure the correct soil type and climate condition are selected.

9. Crop Recommendation

The Crop Recommendation feature helps farmers determine the best crop to grow based on soil and climate conditions.

1. Accessing the Crop Recommendation Feature

  • Go to the Dashboard -> Crop Recommendation section.
  • The crop recommendation form will be displayed.

2. Entering Required Details

  • Soil pH: Enter the pH level of the soil.
  • Temperature: Enter the current temperature in degrees Celsius.
  • Rainfall: Enter the rainfall amount in millimeters (mm).
  • Humidity: Enter the humidity percentage.

3. Getting the Crop Recommendation

  • Click the Get Recommendation button.
  • The system will analyze the entered data and suggest the best crop to grow under the given conditions.

4. Edge Cases & Troubleshooting

No Recommendation Provided?

  • Ensure all fields are correctly filled.
  • Check if the soil pH value is within a valid range (3.5 - 9.0).
  • Temperature, rainfall, and humidity should be reasonable values.

Incorrect or Unexpected Recommendation?

  • Double-check the entered details and submit again.
  • Ensure the values for rainfall and humidity are correct.
